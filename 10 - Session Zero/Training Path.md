# 🚪 Session Zero — Training Path
A route from newcomer to "can run a SPICE exercise and read a RIM map."
1. **The world** → [[ODT Worldbook - Core]] + skim [[Canon & Figures Register]].
2. **The people** → [[Cast Roster]] (read the five full sheets).
3. **The story** → [[THE NUMBER - Season 1 (external)]] (feel the method before learning it).
4. **The method** → [[SPICE Method Brief]] → [[RIM Scoring & LEC Mechanics]].
5. **A monster** → [[BST-S1 NOC Ground Compromise (Cyber)|S1]], then the [[Bestiary Index]] web.
6. **Run it** → [[Module Template]] + [[Elicitation Prompt Tables]].
7. **The data** → [[Data Dictionary]] (how it all becomes a graph).
